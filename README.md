## [KDE] Arch rice dotfiles
### Overview
A minimalist, developer-focused KDE rice built on Arch Linux, featuring:

- Theme: Catppuccin Mocha with custom black adjustments
- Desktop Environment: KDE Plasma
- Color Palette: Dark, purple-ish aesthetic with high-contrast accents

built with the lovely momo ayase and lucy in mind :D
